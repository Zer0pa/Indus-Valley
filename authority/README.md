# Authority Bundle

This folder contains copied authority documents carried into the staged repo so
the front-door truth does not depend on ambient monorepo context.

## Layout

| Folder | Contents |
| --- | --- |
| `review_pack/` | Phase 4/5 summary, verdict, PRD, and search-application surfaces |
| `papers/` | Paper 1 and Paper 2 verdict surfaces |

## Editing rule

Treat these files as copied evidence. Replace them only with new copied source
documents or clearly versioned successors. Do not rewrite them by summary.
